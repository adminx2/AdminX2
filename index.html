Quick Details
What is it?

The purpose of AdminX2 or AdminSquared is to develop an admin panel which is targeted at applications which desire a smaller footprint on some of the general UI. The default Bootstrap elements are largely built with heavy padding and margins. AdminX2 changes many of these styles to make them smaller overall to make them more functional in smaller spaces.

Enhanced Speed/Performance

AdminX2 is built with a minimum of external resources. As an example, while AdminX2 does use an icon font it uses a customized version put together from Fontello which is aimed at minimizing the overal size to improve performance. This means you may need to utilize Fontello or a similar service to build an icon library for your own project but it also ensures your pages won't force more than is needed to be loaded.

Ease of Use

Another very important goal with AdminX2 was to make it as easy to use as possible. As much as possible it sticks to the core functionality found within the Bootstrap library.

Ready to Use JS

The javascript is built to be drop in ready. It is configured to use JS to enhance various elements and if additional libraries are needed those libraries are automatically injected into the JS if not already present so there is no need to concern yourself with adding various JS libraries.

Currently used JS Libraries
  • jQuery
  • Twitter Bootstrap
  • DataTables
Auto Refresh Data
Source
<div class="content" data-refresh="true" data-refresh-speed="5000" data-refresh-target="#activeUsers" data-url="/active-users.php"> </div>
What is it?

Causes a timed event to be created which automatically reloads the data on a set amount of time. In this case 5,000 MS or 5 seconds. data-refresh-target defines a target to be updated. If no data-refresh-target is supplied then the content class div will be replaced.
