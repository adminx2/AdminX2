The previous demonstration is an example of a div being automatically reloaded via ajax through the reload button and by passing a data-url parameter.

Using the data-url attribute in the reload button automatically causes it to re-load the URL specified. It loads pure HTML exactly as it's received and places the result in the content box. 

    <a class="icon-btn reload-btn icon-faded" href="#" data-url="home-box.html">
        <i class="fa icon-ccw-1"></i>
    </a>
